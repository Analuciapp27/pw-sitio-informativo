¿Que es ruby?



                           
                                Se define como uno de los frameworks más utilizados, es lanzado en 2011., es un marco PHP de 
                                código abierto, siendo fácil de entender y modificar. Ofreciendo muchas utilidades que permiten 
                                agilizar el desarrollo de las aplicaciones web. Laravel se enfoca en la calidad del código, 
                                facilidad de mantenimiento y escalabilidad, dando como resultado, permitir realizar proyectos 
                                desde pequeños a grandes o muy grandes. 

                                Se considera fullstack, debido a que ofrece todas las herramientas del backend y el frontend, 
                                contando con una multitud de funciones para poder realizar la implementación de tareas relacionadas 
                                con seguridad, rendimiento e infraestructura. 
                                
                                Este framework es adecuado utilizar para cualquier aplicación web o servicio API que requiera 
                                una lógica de negocio compleja y suportar un volumen elevado de peticiones al servidor. Algunos 
                                ejemplos serían webs con alto tráfico, ecommerce, aplicaciones de gestión interna 
                                para una empresa o un SaaS.

Caracteristicas



                           
                               •	Sistema de ruteo, también RESTful
                               •	Blade, Motor de plantillas
                               •	Peticiones Fluent
                               •	Eloquent ORM
                               •	Basado en Composer
                               •	Soporte para el caché
                               •	Soporte para MVC
                               •	Usa componentes de Symfony
                               •	Adopta las especificaciones PSR-2 y PSR-4

Ventajas





                           
                               • Flexibilidad y agilidad (Faster time to market): funciona de forma modular, ofreciendo 
                                    las funcionalidades y estructuras básicas que necesitas para desarrollar tu aplicación web de forma eficiente.

                               • Arquitectura MVC: Sigue una arquitectura Modelo Vista Controlador (MVC) que separa el código de la 
                                    aplicación en grupos o capas para un desarrollo optimizado.

                               • Ecosistema y documentación: Ofrece un conjunto de herramientas que te ayudan a desarrollar 
                                    algunas de las funcionalidades más complejas de una aplicación web eliminando gran parte de la 
                                    complejidad que suele acompañar a esta tareas.

                               • Motor de temas (Blade) o integración Vue.js (Inertia.js): ofrece el motor de temas Blade, una 
                                    herramienta simple pero poderosa para crear las vistas y componentes en tu aplicación. 
                                    Permitiendo crear código más limpio y eficiente.
                               
                               • Gestión base de datos (Eloquent ORM): Eloquent es una de las herramientas más poderosas de Laravel. 
                                    Ayuda a simplificar enormemente la interacción con la base de datos, permitiendo realizar búsquedas 
                                    sin necesidad de escribir comandos SQL para actualizar o modificar la base de datos.
                               
                               • Seguridad: autenticación y autorización: ofrece multitud de herramientas para adaptarse 
                                    a los requerimientos de tu app. Puedes implementar de forma “simple” el protocolo OAuth2 gracias 
                                    Laravel Passport, con el que ofrecerás los máximos estándares de seguridad a tu servicio API.
                               
                               • Modularidad y amplio repositorio de paquetes: Está desarrollado con la modularidad, 
                                    permitiendo separar tu aplicación en módulos para mejorar la organización de tu código, 
                                    facilitando su mantenimiento y escalabilidad.
                               
                               • Unit testing integrado y programación de tareas: Ofrece soporte para test automáticos, que te 
                                    permitirán realizar esta esencial tarea de forma más eficiente. Permitiéndote simular cualquier 
                                    acción que vaya a realizar tus usuarios en cualquier módulo que componga tu aplicación.
