¿Que es php?



                    
                        Angular es un framework de código abierto que se utiliza para crear aplicaciones web de una sola página. 
                        Es desarrollado por ingenieros de Google en 2012 y se ha logrado mantener desde entonces. 
                        Se adapta a la arquitectura Modelo-Vista-Controlador. Este framework es adecuado para desarrollar 
                        aplicaciones frontend modernas, teniendo complejidad media o elevada. Se desarrollan 
                        aplicaciones JavaScript con el estilo SPA (Single Page Application)
                         o también del estilo PWA (Progressive Web App).

                         AngularJS 
                         – el modelo es el framework, la vista es HTML y el control es JavaScript
                         Ofrece base para poder desarrollar aplicaciones tales como robustas, escalables, y optimizadas, 
                         promoviendo las mejores prácticas al igual que un estilo de modificación 
                         homogéneo y de gran modularidad.
                         Desarrollar aplicaciones con Angular, se realiza a través de TypeScript

Caracteristicas



                    
                        • Uso de DOM regular: Angular utiliza DOM regular, permitiendo una mejor organización conforme 
                          a como va avanzando el desarrollo web. 
                            ¬ DOM (Coument Object Model): documento XML o HTML que contiene una estructura de árbol
                              que esta conformado por nodos, los cuales cada uno representa una parte del documento.

                        • Enlace de datos o data binding: Es un proceso con el que los usuarios pueden manipular 
                             elementos de una página web a través de un navegador, permite una mejora en la 
                             visualización de una página web, sobre todo cuando contiene una gran cantidad de datos.

                        • Compatibilidad móvil y de escritorio: Funciona tanto para el desarrollo de aplicaciones 
                             como de escritorio. Puede ejecutarse en la mayoría de navegadores web.

                        • Velocidad y rendimiento: Cuenta con código de generación, el cual convierte las plantillas 
                             en códigos optimizados. Al igual que da lugar a los sitios que son optimizados y 
                             las aplicaciones angulares se cargan rápidamente.

                        • Productividad: Permite la creación rápida de vistas de interfaz de usuario. Al igual 
                             que con las herramientas de líneas de comandos podemos desarrollar en menor 
                             tiempo y agregar componentes, pruebas al instante.

                        • Enlace bidireccional de datos: Enlaza JavaScript y HTML, teniendo como beneficio 
                             que ambos están sincronizados, así ahorra tiempo para los desarrolladores.

                        • Directivas: Contiene diferentes tipos de directivas muy útiles que puedes usar para 
                             diferentes acciones como vincular el contenido de un elemento HTML a los datos de 
                             aplicación, o especificar que el contenido de un texto debe reemplazarse con una 
                             plantilla.

                        • Pruebas: Hace uso del framework de prueba llamado Jasmine, proporcionando diversas
                             funcionalidades para realizar pruebas en diferentes casos. Admite pruebas 
                             unitarias y de integración.

Ventajas





                    
                        •	Alta calidad de la aplicación

                        •	Desarrollo multiplataforma

                        •	Proceso de desarrollo web más rápido

                        •	Aplicaciones web más ligeras

                        •	Código legible y comprobable

Tipos -> versiones:





                    
                        • Angular 1: Se conoce también como AngularJS, es la versión inicial del framework.
                        
                        • Angular 2: Revisión completa del framework
                        
                        • Angular 4: Paso directamente de angular 2 a angular 4, siendo así la actualización. 
                        
                        • Angular 5: Fue lanzada en 2017. Las mejoras clave incluyen soporte para aplicaciones web progresivas, 
                               un optimizador de construcción y mejoras relacionadas con Material Design.
                        
                        • Angular 6: Fue lanzada en 2018. Se trata de una versión importante centrada menos en el 
                               framework subyacente y más en la cadena de herramientas y en facilitar el movimiento rápido con 
                               Angular en el futuro, como: ng update, ng add, Angular Elements, Angular Material + CDK Components, 
                               Angular Material Starter Components, CLI Workspaces, Library Support, Tree Shakable Providers, Animations
                               Performance Improvements y RxJS v6.
                              
                        • Angular 7: Fue lanzada también en 2018. Estuvo enfocada en actualizaciones 
                               respecto al rendimiento de la aplicación, Angular Material & CDK, desplazamiento 
                               virtual, mejora de la accesibilidad de los selects, ahora soporta la proyección de contenido 
                               usando el estándar web para elementos personalizados, y actualizaciones de dependencias respecto 
                               a Typescript 3.1, RxJS 6.3.

                        • Angular 8: Fue lanzada el 28 de mayo de 2019. Presenta carga diferencial para todo el código de la aplicación, 
                              importaciones dinámicas para rutas perezosas, trabajadores web, soporte de TypeScript.

                        • Angular 9: Fue lanzada el 6 de febrero de 2020. La versión 9 mueve todas las aplicaciones para emplear el 
                              compilador y el tiempo de ejecución de Ivy por defecto. Angular se ha actualizado para funcionar con 
                              TypeScript 3.6 y 3.7. Además de cientos de correcciones de errores, el compilador y tiempo de ejecución Ivy 
                              ofrece numerosas ventajas:
                                Tamaños de paquetes más pequeños
                                Pruebas más rápidas
                                Mejor depuración
                                Mejoras en la vinculación de clases y estilos CSS
                                Mejoras en la comprobación de tipos
                                Mejora de los errores de compilación
                                Mejora de los tiempos de compilación, activando AOT por defecto
                                Mejora de la internacionalización

                        • Angular 10: Fue lanzado el 24 de junio de 2020. Incluye una lista de mejoras como:
                                Nuevo selector de rango de fechas (librería Material UI)
                                Advertencias sobre las importaciones de CommonJS
                                Ajustes opcionales más estrictos
                                Mantenerse al día con el ecosistema
                                Nueva configuración del navegador por defecto
                                Depreciaciones y eliminaciones

                        • Angular 11: Fue lanzado el 11 de noviembre de 2020.
                        • Angular 12: Fue lanzada el 12 de mayo de 2021.
                        • Angular 13:  Fue lanzado el 4 de noviembre de 2021.
